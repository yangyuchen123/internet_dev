# RAG 服务前端测试界面

这是一个简单的前端界面，用于测试 RAG 服务的文件上传和词条检索功能。

## 功能介绍

前端界面包含以下五个标签页：

1. **文件上传** - 支持上传多种格式的文件（PDF、TXT、DOCX、CSV、MD），文件将被处理并添加到知识库
2. **文本输入** - 直接输入文本内容添加到知识库
3. **内容搜索** - 使用向量搜索功能检索相关内容
4. **混合搜索** - 结合关键词和向量搜索的混合检索方法
5. **服务状态** - 显示 RAG 服务的健康状态和模型信息

## 使用方法

### 方法一：使用启动脚本（推荐）

1. 确保已安装所有依赖（见后端 README）
2. 在后端目录运行：
   ```
   start_frontend.bat
   ```
3. 脚本将自动启动后端服务和前端服务
4. 在浏览器中访问 http://localhost:3000

### 方法二：手动启动

1. 确保后端服务已在 http://localhost:8001 运行
2. 在 `frontend` 目录下运行：
   ```
   python -m http.server 3000
   ```
3. 在浏览器中访问 http://localhost:3000

## 注意事项

- 上传大文件时可能需要较长时间处理
- 搜索结果显示相关度分数，分数越高表示内容越相关
- 如果遇到 CORS 错误，请确保后端服务已正确配置 CORS
- 服务状态页面显示当前索引数量和使用的模型

## 浏览器支持

推荐使用最新版本的 Chrome、Firefox、Edge 或 Safari 浏览器。
